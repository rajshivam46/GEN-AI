# 🧠 Deep Learning & Probability Experiments

## 📌 Overview

This repository contains a set of experiments covering fundamental concepts of **Probability, Machine Learning, and Deep Learning**.
The project focuses on practical implementation using Python and popular libraries like NumPy, TensorFlow, and PyTorch.

---

## 🎯 Objectives

* Understand probability distributions and sampling techniques
* Implement Maximum Likelihood Estimation (MLE)
* Train neural networks using backpropagation
* Explore generative models like GAN and VAE
* Evaluate model performance using standard metrics

---

## 🧪 Experiments Included

### 🔹 Experiment 1: Random Sampling

Generate random samples from Normal and Uniform distributions.

### 🔹 Experiment 2: Maximum Likelihood Estimation (MLE)

Estimate mean and variance of a probability distribution.

### 🔹 Experiment 3: Neural Network (MNIST)

Train a neural network using backpropagation and evaluate accuracy.

### 🔹 Experiment 4: Generative Adversarial Network (GAN)

Basic implementation of Generator and Discriminator.

### 🔹 Experiment 5: Flow-Based Model

Simple transformation-based generative model.

### 🔹 Experiment 6: Evaluation Metrics

Implementation of accuracy, precision, recall, and F1-score.

### 🔹 Experiment 7: Model Evaluation

Evaluate model performance on validation dataset.

### 🔹 Experiment 8: Variational Autoencoder (VAE)

Build encoder-decoder architecture for representation learning.

### 🔹 Experiment 9: Autoregressive Model

Basic CNN-based implementation inspired by PixelCNN.

### 🔹 Experiment 10: GAN Loss Analysis

Study adversarial loss and feature matching loss.
